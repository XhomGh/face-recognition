## 使用说明

### 图片人脸识别
1.init图片识别

```bash
python 0_1图片识别先导.py
```
2.人脸关键点检测：
```bash
python 0_2opd人脸关键点检测.py
```
3.人脸关键点标记：
```bash
python 0_3opd标记.py
```


### 视频人脸识别
1.采集训练素材：
```bash
python 1.从相机获取训练素材.py
```
2.提取人脸特征：
```bash
python 2.获取人脸特征.py
```
3.实时识别：
```bash
python 3.相机人脸识别.py
```
